Documentação da Aplicação de Simulação de Recepcção de Hotel

Introdução
Esta documentação fornece uma visão geral da aplicação de simulação de recepção de hotel, incluindo o front-end e o back-end. 
A aplicação permite o cadastro, edição e exclusão de hóspedes, além de fornecer funcionalidades de filtragem por nome.

Tecnologias Utilizadas

Front-end
React.js: Uma biblioteca JavaScript de código aberto para construir interfaces de usuário.
Axios: Uma biblioteca para fazer requisições HTTP a servidores.
Bootstrap: Um framework front-end para desenvolvimento ágil de interfaces web.

Back-end
ASP.NET Core: Um framework de desenvolvimento de software livre e de código aberto desenvolvido pela Microsoft.
MySQL: Um sistema de gerenciamento de banco de dados relacional de código aberto.

Requisitos
Node.js instalado na máquina local para executar o front-end.
ASP.NET Core SDK instalado na máquina local para executar o back-end.
Conexão com a internet para acessar a API back-end e o banco de dados MySQL.
Instalação e Execução
Clone o repositório do projeto para sua máquina local.
Configure o banco de dados MySQL com as configurações fornecidas no arquivo app.settings.
No terminal, navegue até o diretório do front-end e execute o comando npm install para instalar todas as dependências.
Execute o comando npm start para iniciar a aplicação front-end.
No terminal, navegue até o diretório do back-end e execute o comando dotnet run para iniciar a aplicação back-end.
